# PSP Music Downloader

Public releases and one-click updates for PSP Music Downloader.

This repository is intentionally used as a release-only distribution endpoint. Source code, build scripts, bundled tools, assets, signing configuration, and local account state are maintained privately.

## Downloads

Download the latest installer from the [Releases](https://github.com/ZoneJ561/psp-music-downloader/releases/latest) page.

Installed copies of PSP Music Downloader check the public `update.json` release asset and verify the installer SHA-256 hash before launching an update.
