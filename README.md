PSP Music Downloader is designed to make building a PSP music library simple. Search for music, choose complete albums or individual tracks, convert files into PSP-friendly formats, keep your metadata organized, and transfer finished music to a connected PSP.

## Features

### Music Downloads

- Search for artists, albums, singles, and songs inside the app.
- Browse albums and tracks from artist pages.
- Queue multiple albums or songs before starting a download.
- Choose individual tracks from an album.
- Add a direct music URL manually.
- View download progress and estimated completion time.
- Stop downloads and retry failed tracks.
- Sign in through the built-in login window when an authenticated session is required.

### PSP-Friendly Conversion

- Convert music to PSP-compatible MP3 files.
- Choose CBR or VBR encoding.
- Select bitrates up to `256 kbps`.
- Adjust output volume.
- Create ATRAC3 `.at3` files on Windows.
- Remove temporary source files after conversion.

### Metadata And Organization

- Save music as `Artist / Album / 01 Track Name`.
- Preserve album track order.
- Write ID3 tags for title, artist, album, album artist, and track number.
- Add matching album artwork.
- Display explicit and clean release labels.
- Prefer matched, explicit, or clean releases from Settings.
- Run a post-download audit for missing files, duplicate numbers, and release mismatches.

### PSP Transfer

- Detect a connected PSP.
- Transfer finished music to the PSP with one click.
- Keep artist and album folders organized during transfer.

### Library Tools

- Browse downloaded albums from the built-in Library view.
- Scan for duplicate tracks before downloading.
- Clean duplicate files and undo the most recent cleanup.
- Fix or replace metadata.
- Review download history.
- Export a diagnostics report when troubleshooting.

### CD Ripper

- Open the CD Ripper as a separate tool when needed.
- Detect inserted audio CDs.
- Rip a complete CD or selected tracks.
- Look up album information through MusicBrainz, GNUDB, and Discogs.
- Review matches and edit metadata before ripping.

### Updates

- Check for app updates from inside the Windows app.
- Optionally download verified app updates automatically in the background.
- Verify downloaded installers with SHA-256 before installation.
- Restore the previous Windows version when rollback is available.
- Check for and download `yt-dlp` engine updates from Settings.

## Getting Started

1. Download and run the latest installer from the [Releases page](https://github.com/ZoneJ561/psp-music-downloader/releases/latest).
2. Open **Settings** to choose your format, quality, volume, and output folder.
3. Search for music or add a direct URL.
4. Select tracks or albums and choose **Download selected**.
5. Connect your PSP and choose **Transfer to PSP** when your music is ready.
